# 🏛️ L'Agora - Portail Web d'Entraide Étudiante

**L’Agora** est un portail web interactif conçu pour centraliser les besoins de la communauté étudiante tunisienne. Le projet vise à briser l'isolement social et à offrir un soutien concret face aux pressions académiques, financières et psychologiques.

## 📖 Contexte et Problématique

La vie universitaire est souvent marquée par une pression académique forte, l'isolement social ou des difficultés matérielles. Les solutions actuelles étant éparpillées, **L'Agora** propose un espace centralisé et sécurisé où les étudiants peuvent s'exprimer librement, parfois anonymement, pour s'entraider mutuellement.

## 🚀 Objectifs du Projet

*  **Centralisation** : Réunir les besoins et ressources sur une plateforme unique.


*  **Solidarité** : Faciliter le partage de compétences, d'astuces et de dons.


*  **Bien-être** : Offrir un espace d'écoute anonyme pour les préoccupations sensibles.


*  **Expérience Utilisateur** : Proposer une interface fluide et une gestion de données robuste.



## 🛠️ Architecture et Technologies

*  **Frontend** : React.js pour une interface utilisateur dynamique et moderne.


*  **Backend** : Django (Python) pour la logique serveur, les API et la sécurité.


*  **Base de données** : MySQL pour un stockage structuré et fiable.



## 🧩 Modules Développés

*  **M1 - Authentification & Gestion :** Inscription et gestion des comptes utilisateurs (Admin/Étudiant).


*  **M2 - Gestion Publication :** Système complet de création, modification, signalement et modération.


*  **M3 - Gestion Anonymat :** Hachage des identifiants et génération de pseudonymes par le système.


*  **M4 - Gestion Notification :** Notifications internes et envois de mails automatiques.


*  **M5 - Filtrage & Recherche :** Tri multicritères par catégories, tags et dates.


*  **M6 - Gestion de Profil :** Personnalisation et suivi des points de solidarité.

## ⚙️ Installation

1. **Clonez le projet** :
```bash
git clone https://github.com/votre-username/lagora.git

```


2. **Configuration Backend (Django)** :
```bash
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

```


3. **Configuration Frontend (React)** :
```bash
cd frontend
npm install
npm start

```
